# 2026-01-26: 프론트엔드 작업 일정 상세화

## 작업 배경
GO ALMOND 서비스 기획서를 기반으로 프론트엔드 개발 일정 및 상세 작업 내용을 산출

## 서비스 개요
- **비전**: Going to America to Learn and Make One's Dream come true!
- **범위**: 미국 유학 전 과정 통합 플랫폼 (상담-검색-신청-입학-비자-이주-정착-진로)
- **핵심 프로그램**: 
  - 직업학교 (Vocational School)
  - 커뮤니티 컬리지 (Community College)
  - 일반 대학/대학원 (University/Graduate)
  - 초등학교 참관 (Elementary School)

## 주요 변경사항

### 1. 문서 구조 확장
- **서비스 개요**: 서비스 비전, 핵심 가치, 서비스 대상 명시
- **디자인 시스템**: 
  - 컬러 팔레트 (밝은 녹색/밝은 회색/흰색/오렌지)
  - 기구축 UI 컴포넌트 목록 정리
- **화면 구조 및 라우팅**: 인증 전/후 페이지 라우팅 맵 상세화 (20개 이상 페이지)
- **전체 일정 요약**: 표 형식으로 9개 Phase 정리 (16~17주)

### 2. 단계별 상세 작업 내용

#### Phase 0: 설계/기반 구축 (1주)
- 정보 아키텍처 설계
- 디자인 토큰 적용
- 공통 레이아웃 구축
- API 클라이언트 설정

#### Phase 1: 인증 전 페이지 (2주)
- 랜딩 페이지 (Hero, 서비스 소개, 성공 사례)
- 회원가입 (약관 동의, 단계별 플로우)
- 로그인 (이메일/OAuth)

#### Phase 2: 필수 정보 입력 시스템 (2주)
- 학교 정보 입력
- 개인 정보 입력
- 희망 유학 케이스 입력
- Form Wizard + 유효성 검사

#### Phase 3: AI 추천/맞춤 리포트 (2주)
- 매칭 지표 시각화 (퍼센트 + 6개 지표)
- 추천 리스트 (필터링/정렬)
- 추천 상세 + 리포트 (PDF/공유)

#### Phase 4: 4개 핵심 프로그램 페이지 (2주)
- 직업학교 프로그램
- 커뮤니티 컬리지 프로그램
- 일반 대학/대학원 프로그램
- 초등학교 참관 프로그램

#### Phase 5: 검색 및 학교 상세 (1.5주)
- 통합 검색 (학교/전공/직업/지역)
- 학교 상세 페이지 (보관/지원/문의)

#### Phase 6: 문서 표준화 (1.5주)
- 지원 서류 표준화 (페이지 뷰/PDF)
- 비자 서류 표준화 (페이지 뷰/PDF)

#### Phase 7: 미국 생활 정보 허브 (1주)
- 6개 카테고리 (숙소/운전면허/차량/보험/OPT)

#### Phase 8: 로그인 후 메인/마이페이지 (2주)
- 대시보드 (프로그램 요약/신청 현황/AI 추천/인기 학교)
- 마이페이지 (정보 수정/지원 현황/보관함)

#### Phase 9: QA/성능/접근성 (1~2주)
- 반응형/접근성/성능 최적화/보안 점검

### 3. 페이지별 작업 시간 요약
- **총 작업일**: 77일 (약 15.4주)
- **우선순위 분류**: 높음/중간/낮음

### 4. 기술 스택
- **현재 프로젝트**: React 18, TypeScript, Vite, Tailwind CSS, React Router, Axios, Zod, Vitest
- **추가 필요 라이브러리**:
  - Recharts (매칭 지표 시각화)
  - react-pdf (PDF 생성)
  - Google Maps API (지역 환경)
  - React Hook Form (복잡한 폼 관리)
  - react-dropzone (파일 업로드)
  - Swiper (캐러셀)
  - react-markdown (가이드 렌더링)

### 5. 리스크 및 의존성
- **백엔드 의존성**: API 스펙, AI 추천 알고리즘, 학교 DB, 문서 템플릿, 검색 엔진
- **기능 복잡도**: PDF 생성, 공유 기능, 챗봇, 지도 연동
- **디자인 확정**: 디자인 시스템, 컴포넌트 갤러리, 에셋 준비

### 6. 다음 단계 (Action Items)
1. 백엔드 API 스펙 정의 회의
2. 디자인 시스템 확정 (Hex 코드, Typography 스케일)
3. 외부 서비스 계약 (Google Maps API, PDF 생성, 챗봇)
