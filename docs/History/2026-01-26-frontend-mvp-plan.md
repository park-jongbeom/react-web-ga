# 2026-01-26: 프론트엔드 MVP 계획 수립

## 작업 배경
백엔드 AI 매칭 MVP 계획을 참고하여 프론트엔드도 **핵심 기능만 추린 MVP 버전**으로 일정 재산정

## MVP 전략
- **핵심 목표**: AI 매칭 엔진을 체험할 수 있는 최소 기능
- **기간**: 6주 (60시간, 주당 10시간)
- **협업 전략**: 백엔드 Mock API(Week 1) 완료 후 즉시 시작 → 병렬 개발

## 기존 계획 vs MVP 계획 비교

### 1. 일정
| 구분 | 기존 계획 | MVP 계획 |
|------|----------|---------|
| 기간 | 16~17주 | **6주 (63% 단축)** |
| 총 시간 | 77일 (약 308시간) | **60시간 (81% 단축)** |
| 산정 기준 | 1명, 주 5일 | 1명, 주당 10시간 (월~목) |

### 2. 페이지 수
| 구분 | 기존 계획 | MVP 계획 |
|------|----------|---------|
| 총 페이지 | 20개 이상 | **8개 (60% 축소)** |

### 3. 핵심 기능 비교

#### ✅ MVP 포함 기능 (7개)
1. **회원가입/로그인** (기존 유지)
2. **필수 정보 입력** (3단계 Wizard) (기존 유지)
3. **AI 매칭 실행 + 결과 표시** (6대 지표 Radar Chart) (기존 유지)
4. **추천 학교 리스트** (Top 5) (기존 단순화)
5. **학교 상세 보기** (설명 가능한 추천 이유 포함) (기존 단순화)
6. **지원 관리** (보관/지원 상태) (기존 단순화)
7. **간단한 대시보드** (매칭 현황 + 지원 현황) (기존 단순화)

#### ❌ MVP 제외 기능 (Phase 2로 이동)
1. **4개 프로그램 별 상세 페이지** (Vocational/Community/University/Elementary)
2. **문서 표준화** (지원/비자 서류 뷰어 + PDF 다운로드)
3. **미국 생활 정보 허브** (6개 카테고리)
4. **복잡한 검색** (학교/전공/직업/지역 통합 검색)
5. **PDF 생성/공유 기능** (이메일/메신저/SNS)
6. **챗봇 문의**
7. **지도 연동** (Google Maps API)
8. **파일 업로드** (문서 관리)
9. **캐러셀** (성공 사례 슬라이드)
10. **마크다운 렌더링** (가이드 콘텐츠)

## 주요 변경사항

### 1. Epic 구조 단순화 (9개 → 6개)
- **기존**: Phase 0~9 (16~17주)
- **MVP**: Epic 1~6 (6주)
  - Epic 1: 인증 + 프로필 입력 (1/3)
  - Epic 2: 프로필 입력 (2/3) + Mock 연동
  - Epic 3: 매칭 결과 UI + 학교 리스트
  - Epic 4: 실제 API 연동 + 학교 상세
  - Epic 5: 지원 관리 + 대시보드
  - Epic 6: 통합 테스트 + 최적화

### 2. 백엔드 병렬 개발 전략
```
백엔드 Week 1 → Mock API 제공
              ↓
프론트 Week 1~3 → Mock API로 UI 개발
              ↓
백엔드 Week 4 → 실제 매칭 API 완성
              ↓
프론트 Week 4~5 → 실제 API 연동 + 지원 관리
              ↓
백엔드 Week 6 → 보안 + 모니터링
프론트 Week 6 → 통합 테스트 + 최적화
```

### 3. 컴포넌트 재사용 전략
- **기구축 컴포넌트 재사용률: 약 60%**
- 이미 완성된 Base UI 컴포넌트 활용
  - `BaseButton`, `BaseInput`, `BaseCard`
  - `BaseHeading`, `BaseText`
  - `BaseSection`, `BaseContainer`
  - `AuthContext`, `ProtectedRoute`, `ErrorBoundary`

### 4. 페이지별 작업 시간 (총 60시간)
| 페이지/기능 | 시간 |
|------------|------|
| 회원가입/로그인 | 4h |
| 프로필 입력 (3단계) | 11h |
| 매칭 실행 + 결과 | 6h |
| 학교 리스트 | 4h |
| 학교 상세 | 6h |
| 지원 관리 | 3h |
| 보관함 | 2h |
| 대시보드 | 5h |
| 통합 테스트 | 3h |
| 반응형 테스트 | 2h |
| 성능 최적화 | 3h |
| 보안 점검 | 2h |
| Mock API 연동 | 5h |
| 실제 API 전환 | 2h |
| 기타 (버퍼) | 2h |
