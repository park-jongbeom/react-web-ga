# 📋 Data Definition (Global Study-Abroad Platform)

## 1. 개인 프로필 (Personal - 암호화 대상)
- Legal Name (성/이름 - 여권 일치)
- 여권 정보 (번호, 만료일)
- 연락처 (국적, 주소, 전화번호)

## 2. 학업 데이터 (Academic)
- 최종 학력 정보 (학교명, 졸업일)
- GPA (원본 점수 및 4.0 환산 기준)
- 공인 영어 성적 (TOEFL, IELTS, Duolingo 등)

## 3. 재정 및 비자 (Finance / Visa)
- 유학 자금 출처 및 보증인 정보
- 잔고 증명 정보 및 증빙 서류 경로
- 희망 비자 타입 (F-1, M-1 등) 및 SEVIS ID

## 4. 지원 정보 (Application)
- 입학 희망 시기 (Target Term)
- 희망 전공 (STEM 여부 확인용)
- 관심 학교 리스트 (Wishlist)
